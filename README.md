# TODO WEB APPLICATION

This is a full-stack Todo application built with React and Typescript.
It supports the user to create, edit, and delete a to-do list.
This project has separate folders for backend and frontend.


## TECH STACK
### FrontEnd

- React(with Vite)
- Typescript
- React Router
- React Query
- Zustand
- Axios
- TailwindCSS {This one I just assumed, and to make the site look appealing with good UI, I used global CSS for it}

### BackEnd

- Node.js
- Express.js
- MongoDB + Mongoose
- JSON Web Tokens (JWT)
- bcrypt
